mavenWebApp
===========

Create a web app that exposes a set of restfull services using apache cfx and spring.
The project is created using maven maven-archetype-webapp archetype
Pre-requisities
Eclipse J2EE (3.7 or later)
JDK 1.6 (or later)
mvn 3.0.4
